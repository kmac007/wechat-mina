前不久ofo的小程序重新上线了，和摩拜一样，使用微信小程序来扫码租车十分方便。十分契合小程序“用完即走”的理念。我体验了一番，小程序的流畅程度是优于普通webapp的，虽然暂时还不了解小程序的底层知识，但我觉得小程序是值得学习的。另外，豆瓣提供的开放API十分友好，因此我通过查看相应的文档写下了这个小程序。
